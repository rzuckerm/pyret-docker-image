# pyret-docker-image

Docker image for [Pyret programming language](https://pyret.org/docs/latest/):

- rzuckerm/pyret:`<version>-<tag>`

where:

- `<version>` is the [Pyret version](PYRET_VERSION)
- `<tag>` is the current GitHub tag without the "v"

The docker image can be found [here](https://hub.docker.com/r/rzuckerm/pyret).
